Cadastro de Ativos

Este projeto é uma aplicação web desenvolvida para gerenciar o cadastro de ativos de uma instituição. Com ele, é possível registrar e acompanhar ativos como computadores, monitores e outros equipamentos de TI. A aplicação oferece uma interface intuitiva para adicionar, editar e excluir ativos, além de funcionalidades para consultar o histórico de cada item. O sistema foi construído utilizando tecnologias como Flask, SQLite, HTML, CSS e JavaScript.

Controle de Itens da Casa

Este projeto é uma solução web para ajudar na organização dos itens domésticos. Com ele, os usuários podem catalogar e gerenciar itens de diferentes categorias, como produtos de limpeza, alimentos e utensílios. A aplicação permite verificar o status dos itens, saber onde estão localizados e receber alertas sobre itens que precisam ser repostos. Desenvolvido com Flask, HTML, CSS e JavaScript, este projeto visa facilitar a gestão doméstica de maneira eficiente e prática.
